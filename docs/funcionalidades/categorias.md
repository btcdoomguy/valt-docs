# Categorias

As categorias permitem classificar suas transações, facilitando a análise de para onde está indo seu dinheiro e de onde ele vem.

## Visão Geral

Categorias são etiquetas que você atribui às suas transações. Elas são essenciais para:

- Entender seus padrões de gastos
- Gerar relatórios significativos
- Identificar oportunidades de economia
- Organizar suas finanças

## Categorias Padrão

O Valt já vem com categorias pré-configuradas para você começar rapidamente:

### Despesas

- Alimentação
- Moradia
- Transporte
- Saúde
- Educação
- Lazer
- Vestuário
- Serviços

### Receitas

- Salário
- Freelance
- Investimentos
- Outros

Você pode usar essas categorias ou personalizá-las conforme sua necessidade.

## Gerenciando Categorias

### Acessando o Gerenciador

1. Clique no menu de configurações
2. Selecione **Gerenciar Categorias**
3. O painel de categorias será aberto

### Criando uma Nova Categoria

1. Clique no botão **+** ou **Nova Categoria**
2. Preencha:

| Campo | Descrição |
|-------|-----------|
| **Nome** | Nome da categoria (ex: "Supermercado") |
| **Ícone** | Ícone visual |
| **Cor** | Cor para identificação |

3. Clique em **Salvar**
4. 
### Editando uma Categoria

1. Na lista de categorias, clique na categoria desejada
2. Clique no ícone de edição
3. Modifique os campos
4. Clique em **Salvar**

### Excluindo uma Categoria

!!! warning "Atenção"
    O Valt não permite excluir categorias já utilizadas em alguma transação.

1. Clique na categoria
2. Clique em **Excluir**
3. Confirme a exclusão

## Subcategorias

O Valt suporta categorias hierárquicas, ou seja, categorias dentro de categorias.

Para criar uma subcategoria, basta arrastar a subcategoria filha para dentro de uma categoria pai.

### Exemplo de Hierarquia

```
Alimentação
├── Supermercado
├── Restaurantes
├── Delivery
└── Padaria

Transporte
├── Combustível
├── Uber/99
├── Transporte Público
└── Manutenção do Carro
```

## Escolhendo Ícones

Cada categoria pode ter um ícone associado:

1. Ao criar/editar uma categoria, clique no campo **Ícone**
2. Uma biblioteca de ícones será exibida
3. Use a busca para encontrar ícones (ex: "food", "car", "home")
4. Clique no ícone desejado

### Dicas para Ícones

- Use ícones intuitivos que representem a categoria
- Mantenha consistência visual
- Ícones facilitam a identificação rápida nas listas

## Cores das Categorias

As cores ajudam na identificação visual:

1. Clique no campo **Cor**
2. Selecione uma cor da paleta

## Boas Práticas

### Estrutura Recomendada

**Não crie categorias demais**. Muitas categorias dificultam a análise. Uma estrutura de 8-12 categorias principais é suficiente para a maioria das pessoas.

**Exemplo de estrutura equilibrada**:

```
Receitas
├── Salário
├── Freelance
├── Investimentos
└── Outras Receitas

Despesas
├── Moradia
├── Alimentação
├── Transporte
├── Saúde
├── Lazer
├── Educação
└── Outros

Transferências
└── Entre Contas
```

### Categorização Consistente

- Defina regras claras para cada categoria
- Exemplo: "Alimentação" é para comida consumida, "Mercado" pode incluir itens de limpeza
- Siga suas próprias regras consistentemente

## Uso nos Relatórios

As categorias são fundamentais para os relatórios:

### Gastos por Categoria

O relatório de **Gastos por Categoria** mostra:
- Quanto você gastou em cada categoria
- Porcentagem do total
- Comparativo entre períodos

### Análises

Com uma boa categorização, você pode responder:

- "Quanto gasto com alimentação por mês?"
- "Meus gastos com transporte aumentaram?"
- "Qual categoria representa a maior parte das minhas despesas?"

## Alterando Categoria de Múltiplas Transações

Se você precisa recategorizar várias transações:

1. Selecione múltiplas transações na lista de transações segurando Shift ou Ctrl
2. Abra o menu Editar Todas
3. Marque a opção de alterar a categoria e escolha a nova categoria

Isso é útil quando:
- Você reorganizou suas categorias
- Percebeu que categorizou errado
- Quer mesclar várias categorias
